ID y class

Esto nos sirve para poder poner identificar mejor las cosas. Esto nos sera muy util para luego CSS

El ID tiene que ser unico, es su "dni"

Mientras que class nos sirve para poner una identificacion a un grupo de elementos. Por ejemplo la clase "libros"

Ejemplo de un colegio: El ID seria cada alumno, no su nombre y apellidos ya que pueden coincidir, seria su numero de alumno por ejemplo.
Class ya seria la propia clase de los alumnos, o distintas etiquetas para cada alumno; "deportistas", "aplicados", "vagos"...
