Multimedia Video

Video

Atributos importantes para video

Heigth
Esto sirve para limitar la altura del video y reserva ya un espacio para la web. Esto lo damos en pixeles (px)
Width
Ocurre lo mismo que con las imagenes, una vez definido el Height no hace falta definir el ancho para que no se nos deforme el video
Controls
Muestra botones de control para el video. Podemos crear unos controles nuevos con JS, pero estos no podremos modificarlos
Autoplay
Reproduce el video directamente en cuanto cargue
Loop
Una vez que el video acabe volvera a reproducirse desde el principio. Independientemente del autoplay
Muted
Seteamos el volumen a 0, por lo que el usuario no podra escucharlo. Podremos modificar ese valor mas tarde con JS
Poster
Una imagen que se muestra mientras el video no empieza a reproducirse. Tambien se pueden poner gifs para hacer un loading

Audio

Atributos

Tiene los mismos atributos que los videos, siempre sera bueno añadirle el atributo controls para poder visualizar el reproductor de audio Lo unico que no comparte es el atributo de poster

iFrame

Permite crear una ventana dentro de la propia pagina para previsualizar cualquier tipo de archivo
