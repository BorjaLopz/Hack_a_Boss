    Diferencias listas ordenadas y listas desordenadas

    Las listas ordenadas tienen algo mas de informacion, ya que tienen un orden, una prioridad. Por ejemplo si queremos hacer una lista de nuestras peliculas favoritas utilizaremos un lista ordenada. Si por el contrario queremos hacer una lista de las peliculas que hemos visto en el ultimo a√±o, podremos utilizar una lista desordenada perfectamente.

    Ver el ejemplo de la receta de la tortilla de patatas

    Diferencias listas desordenadas y menu

    Mientras que una lista tiene elementos como texto para leer y nada mas, en un menu aparece una lista de articulos que son interactuables como botones y enlaces
